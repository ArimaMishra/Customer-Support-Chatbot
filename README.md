# 🤖 Customer Support Chatbot

A simple rule-based Customer Support Chatbot developed in Python using Regular Expressions (Regex). The chatbot interacts with users and responds to common customer queries such as product information, technical support, and return policy.

---

## 📌 Features

- Greets the user
- Stores user's name
- Answers product-related queries
- Provides technical support information
- Explains return policy
- Handles unknown queries politely
- Exits the conversation using commands like:
  - bye
  - exit
  - quit
  - goodbye

---

## 🛠 Technologies Used

- Python
- Regular Expressions (re)
- Random Module
- Jupyter Notebook

---

## ▶️ How to Run

1. Install Python or Anaconda.
2. Open Jupyter Notebook.
3. Open `Customer_Support_Chatbot.ipynb`
4. Run all cells.
5. Start chatting with the bot.
